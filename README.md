# 📝  Dotfiles

Personal dotfiles and scripts.

## Install config files
```bash
$ git clone git@github.com:goldenson/dotfiles.git
$ ln -s /Users/goldenson/dotfiles/.zshrc ~/.zshrc
```

## Use ctags

```bash
$ brew install ctags
$ ctags -R -f .tags
```
